ğŸ” NEW API ENDPOINTS
================================================================================

1. GET /gs1/search/gtin?gtin=XXX
   Purpose: Search GS1 by barcode
   Params: gtin (required)
   Returns: Product data with image
   
2. GET /gs1/search/brand?brand=X&productName=Y
   Purpose: Search GS1 by brand
   Params: brand, productName (required)
   Returns: Array of matching products
   
3. GET /gs1/brands
   Purpose: Get common brands list
   Params: None
   Returns: Array of brand names
   
4. POST /products/:id/gs1-lookup
   Purpose: Link product to GS1
   Body: {gtin: "string"}
   Returns: Updated product with GS1 data
